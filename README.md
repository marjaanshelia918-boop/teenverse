# 🌟 TeenVerse - Social Platform for Teenagers

Welcome to **TeenVerse**, a vibrant social platform designed specifically for teenagers to explore, learn, create, and connect with peers worldwide!

## 📋 Project Overview

TeenVerse is a modern web application that provides a safe, engaging space for teens to:
- 🎮 **Discover and Play Games** - Connect with gamers and play trending games
- 🎵 **Share Music** - Build playlists and discover new artists
- 📚 **Learn Together** - Access free coding tutorials, science lessons, and study guides
- 🎨 **Showcase Creativity** - Share artwork, photography, and videos
- 👥 **Build Community** - Join discussions, groups, and events
- 🔍 **Explore New Interests** - Discover content, creators, and communities

## 🚀 Getting Started

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marjaanshelia918-boop/teenverse.git
   cd teenverse
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server: `python -m http.server 8000`

### Enable GitHub Pages

1. Go to your repository: [https://github.com/marjaanshelia918-boop/teenverse](https://github.com/marjaanshelia918-boop/teenverse)

2. Click on **Settings** in the top menu

3. Scroll down to the **Pages** section

4. Under "Build and deployment":
   - Select **Source:** `Deploy from a branch`
   - Select **Branch:** `main`
   - Select **Folder:** `/ (root)`
   - Click **Save**

5. Wait 1-2 minutes for deployment

6. Your site will be live at: `https://marjaanshelia918-boop.github.io/teenverse/`
